# Polar Design - Creative Web & Multimedia Agency
Polar Design คือ Digital Solutions Agency ที่มุ่งเน้นการยกระดับแบรนด์ผ่านการออกแบบเว็บไซต์ที่ล้ำสมัยและงานมัลติมีเดียครบวงจร <br>
(Video, Graphic, Animation) เราเปลี่ยนไอเดียให้กลายเป็นประสบการณ์ดิจิทัลที่จับต้องได้และตอบโจทย์ธุรกิจอย่างแท้จริง

## Key Features
- Responsive Design: รองรับการใช้งานสมบูรณ์แบบในทุกอุปกรณ์ (Desktop, Tablet, Mobile)
- Modern UI/UX: เน้นความสวยงาม เรียบง่าย แต่ทรงพลังตามมาตรฐานการออกแบบยุคใหม่
- Performance Optimized: ปรับแต่งความเร็วในการโหลดหน้าเว็บ เพื่อประสบการณ์ที่ดีของผู้ใช้
- Interactive Multimedia: การนำเสนองานมัลติมีเดียที่น่าสนใจและลื่นไหล

## Objectives & Vision
- Business Centric: พัฒนาเว็บไซต์ที่ช่วยสร้างภาพลักษณ์และยอดขายให้แก่ธุรกิจ
- High-Quality Multimedia: สื่อสารแบรนด์ผ่านงานกราฟิกและแอนิเมชันคุณภาพสูง
- User Experience First: ให้ความสำคัญกับความง่ายในการใช้งาน (Usability) เป็นอันดับหนึ่ง

## Getting Started
### 1. Clone โปรเจกต์
git clone https://github.com/namnueng32371/polar-design.git

### 2. เข้าไปที่ Folder
cd polar-design

### 3. สามารถดับเบิลคลิกที่ไฟล์ `index.html` เพื่อดูหน้าเว็บได้ทันที
   - หรือหากใช้ VS Code แนะนำให้คลิกขวาที่ไฟล์ `index.html` แล้วเลือก **"Open with Live Server"** เพื่อดูการแสดงผลแบบ Real-time
